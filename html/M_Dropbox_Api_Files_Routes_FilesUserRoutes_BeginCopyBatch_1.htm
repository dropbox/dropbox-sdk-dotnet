autorename (Optional)
Type: SystemBoolean
If there's a conflict with any file, have the Dropbox server try to autorename that file to avoid the conflict.
allowSharedFolder (Optional)
Type: SystemBoolean
If true, CopyBatchAsync(RelocationBatchArg) will copy contents in shared folder, otherwise RelocationErrorCantCopySharedFolder will be returned if FromPath contains shared folder. This field is always true for MoveBatchAsync(RelocationBatchArg).
allowOwnershipTransfer (Optional)
Type: SystemBoolean
Allow moves by owner even if it would result in an ownership transfer for the content being moved. This does not apply to copies.
