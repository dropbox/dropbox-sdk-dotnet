name

The last component of the path (including extension). This never contains a slash.
id
Type: SystemString
A unique identifier for the folder.
pathLower (Optional)
Type: SystemString
The lowercased full path in the user's Dropbox. This always starts with a slash. This field will be null if the file or folder is not mounted.
pathDisplay (Optional)
Type: SystemString
The cased path to be used for display purposes only. In rare instances the casing will not correctly match the user's filesystem, but this behavior will match the path provided in the Core API v1, and at least the last path component will have the correct casing. Changes to only the casing of paths won't be returned by ListFolderContinueAsync(ListFolderContinueArg). This field will be null if the file or folder is not mounted.

sharedFolderId (Optional)

sharingInfo (Optional)
Type: Dropbox.Api.FilesFolderSharingInfo
Set if the folder is contained in a shared folder or is a shared folder mount point.
propertyGroups (Optional)
Type: System.Collections.GenericIEnumerablePropertyGroup
Additional information if the file has custom properties with the property template specified. Note that only properties associated with user-owned templates, not team-owned templates, can be attached to folders.
