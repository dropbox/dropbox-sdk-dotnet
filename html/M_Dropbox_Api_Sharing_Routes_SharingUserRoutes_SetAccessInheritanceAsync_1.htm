SharingUserRoutesSetAccessInheritanceAsync Method (String, AccessInheritance)

Change the inheritance policy of an existing Shared Folder. Only permitted for shared folders in a shared team root.

If a ShareFolderLaunchAsyncJobId is returned, you'll need to call CheckShareJobStatusAsync(PollArg) until the action completes to get the metadata for the folder.
