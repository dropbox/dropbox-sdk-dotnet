SharedFolderJoinPolicy

Policy governing which shared folders a team member can join.

SharedFolderJoinPolicyFromAnyone

Team members can join any shared folder, including those shared by users outside the team.

SharedFolderJoinPolicyFromTeamOnly

Team members can only join folders shared by teammates.

SharedFolderJoinPolicyOther

SharedFolderMemberPolicy

Policy governing who can be a member of a folder shared by a team member.

SharedFolderMemberPolicyAnyone

Anyone can be a member of a folder shared by a team member.

SharedFolderMemberPolicyOther

The other object

SharedFolderMemberPolicyTeam

Only a teammate can be a member of a folder shared by a team member.

SharedLinkCreatePolicy

Policy governing the visibility of newly created shared links.

SharedLinkCreatePolicyDefaultPublic

By default, anyone can access newly created shared links. No login will be required to access the shared links unless overridden.

SharedLinkCreatePolicyDefaultTeamOnly

By default, only members of the same team can access newly created shared links. Login will be required to access the shared links unless overridden.

SharedLinkCreatePolicyOther

The other object

SharedLinkCreatePolicyTeamOnly

Only members of the same team can access newly created shared links. Login will be required to access the shared links.
