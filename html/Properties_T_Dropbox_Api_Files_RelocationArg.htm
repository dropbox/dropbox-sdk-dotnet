The RelocationArg type exposes the following members.

Properties
  NameDescription
AllowOwnershipTransfer

Allow moves by owner even if it would result in an ownership transfer for the content being moved. This does not apply to copies.

AllowSharedFolder

If true, CopyAsync(RelocationArg) will copy contents in shared folder, otherwise RelocationErrorCantCopySharedFolder will be returned if [!:FromPath] contains shared folder. This field is always true for MoveAsync(RelocationArg).

Autorename

If there's a conflict, have the Dropbox server try to autorename the file to avoid the conflict.
