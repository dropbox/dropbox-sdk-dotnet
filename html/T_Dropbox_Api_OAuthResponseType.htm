Token0 This represents the OAuth 2.0 token or implicit grant flow. The server will return the bearer token via the redirectUri callback, rather than requiring your app to make a second call to a server. This is useful for pure client-side apps, such as mobile apps or JavaScript-based apps.
Code1 This represents the OAuth 2.0 code flow. The server will return a code via the redirectUri callback which should be converted into a bearer token using the ProcessCodeFlowAsync(String, String, String, String, HttpClient) method. This is the recommended flow for apps that are running on a server.
