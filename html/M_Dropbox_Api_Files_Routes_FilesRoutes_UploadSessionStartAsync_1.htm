[This is preliminary documentation and is subject to change.]

Upload sessions allow you to upload a single file using multiple requests. This call starts a new upload session with the given data. You can then use UploadSessionAppendAsync(UploadSessionCursor, Stream) to add more data and UploadSessionFinishAsync(UploadSessionFinishArg, Stream) to save all the data to a file in Dropbox.

A single request should not upload more than 150 MB of file contents.
